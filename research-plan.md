## 2. スコアリング基準

各習慣を以下の3軸で1-5点で評価し、総合スコアを算出する。

### 2-1. エビデンスの信頼度（Evidence Reliability）: 1-5
| スコア | 基準 |
|--------|------|
| 5 | 複数のRCT + メタ分析が存在。効果の方向性が一貫 |
| 4 | 複数のRCTまたは大規模観察研究。概ね一貫した結果 |
| 3 | 少数のRCTまたは複数の準実験的研究。結果にやや不一致あり |
| 2 | 観察研究・相関研究が中心。因果推論が弱い |
| 1 | 理論的根拠のみ、または事例研究レベル |

### 2-2. 創造性向上の効果量（Effect Size）: 1-5
| スコア | 基準 |
|--------|------|
| 5 | 大きな効果量（d ≥ 0.8 または 50%以上の向上） |
| 4 | 中〜大の効果量（d = 0.5-0.8 または 30-50%の向上） |
| 3 | 中程度の効果量（d = 0.3-0.5 または 15-30%の向上） |
| 2 | 小さな効果量（d = 0.2-0.3 または 10-15%の向上） |
| 1 | 効果量が不明確、または非常に小さい |

### 2-3. 取り入れやすさ（Ease of Adoption）: 1-5
| スコア | 基準 |
|--------|------|
| 5 | 今日からすぐ始められる。コスト・時間がほぼ不要 |
| 4 | 軽微な準備で始められる。週に数回、短時間で実践可 |
| 3 | ある程度の学習・投資が必要。継続にやや努力を要する |
| 2 | 相当な準備・投資・環境変化が必要 |
| 1 | 大幅な生活変化が必要（海外移住等） |

### 2-4. 総合スコア計算方法
```
総合スコア = (信頼度 × 0.35) + (効果量 × 0.40) + (取り入れやすさ × 0.25)
```
※ 効果量を最も重視し、次に信頼度、最後に取り入れやすさ

## 3. リサーチ実行計画

### エージェント構成

| エージェント | 担当 | 役割 |
|------------|------|------|
| Research Agent 1 | カテゴリ A-E（10習慣） | 複数論文の調査、効果量の特定、信頼度評価 |
| Research Agent 2 | カテゴリ F-J（10習慣） | 同上 |
| Research Agent 3 | カテゴリ K-O（10習慣） | 同上（新カテゴリN,Oの探索含む） |
| Quality Check Agent | 全体 | 結果の整合性チェック、スコアリングの妥当性検証、漏れの指摘 |

### 各リサーチエージェントへの調査指示
1. 各習慣について最低2-3本の査読付き論文を特定する
2. 効果量（Cohen's d、%向上、オッズ比等）を可能な限り抽出する
3. 研究デザイン（RCT、準実験、観察、メタ分析）を記録する
4. 矛盾する知見や限界点を明記する
5. スコアリング基準に基づく仮評価を行う

### 品質チェックエージェントへの検証指示
1. 各習慣のスコアが基準と整合しているか
2. 論文の引用が妥当か（存在確認、内容との一致）
3. 同じ効果を主張する習慣間の重複・整理が必要か
4. 全体として見落とされている重要なエビデンスがないか
5. スコアの分布が偏りすぎていないか
